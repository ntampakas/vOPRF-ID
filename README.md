# 🔐 vOPRF-ID

## 📝 Overview

vOPRF-ID is a monorepo containing implementations for secure nullifier generation based on verifiable Oblivious Pseudorandom Functions (vOPRF).

To host the [docs](./docs/) locally:
```bash
cd docs
mdbook serve
```
